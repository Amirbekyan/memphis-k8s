Memphis UI can be accessed on the following DNS name from within your cluster: memphis-ui.{{ .Release.Namespace }}.svc.cluster.local                                                                                        
To access Memphis from localhost, run the below commands:                                                                                    
  1. kubectl port-forward service/memphis-ui 9000:80 --namespace {{ .Release.Namespace }} > /dev/null &
  2. kubectl port-forward service/memphis-cluster 7766:7766 --namespace {{ .Release.Namespace }} > /dev/null &
  3. kubectl port-forward service/control-plane 6666:6666 6667:80 --namespace {{ .Release.Namespace }} > /dev/null &

Dashboard: http://localhost:9000
CLI: http://localhost:9000

## Secrets ##
UI/CLI/SDK root username        - root
UI/CLI root Password        - kubectl get secret memphis-creds -n {{ .Release.Namespace }} -o jsonpath="{.data.ROOT_PASSWORD}" | base64 --decode
SDK Connection Token  - kubectl get secret memphis-creds -n {{ .Release.Namespace }} -o jsonpath="{.data.CONNECTION_TOKEN}" | base64 --decode
